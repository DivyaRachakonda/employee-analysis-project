# Employee Analysis Project

Employee data analysis project using Python, MySQL, Pandas, and Matplotlib.

## Features
- Average salary by department
- Employee count by department
- Average salary by city
- Top 10 employees by salary analysis

## Technologies Used
- Python
- MySQL
- Pandas
- Matplotlib

## Project Description
This project connects Python with a MySQL database to analyze employee data and visualize insights using charts. It helps understand salary trends across departments and cities.
